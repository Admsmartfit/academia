Como funciona
XP Disponivel

O aluno acumula XP nos ultimos 3 meses. XP mais antigo sai da janela de conversao, mas continua valendo para o ranking.

Conversao Automatica

Se ativada, o sistema converte automaticamente quando o aluno atinge o XP necessario (respeitando cooldown).

Validade dos Creditos

Creditos gerados tem data de expiracao. O sistema usa FIFO: primeiro a vencer, primeiro a usar.
